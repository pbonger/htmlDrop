# HTML Drop — Installation

Right-click any `.html` file to share it on HTML Drop. Works in WebStorm and macOS Finder.

---

## WebStorm Plugin

1. Open WebStorm → **Settings → Plugins**
2. Gear icon → **Install Plugin from Disk**
3. Select `html-drop-plugin-*.zip` from this folder
4. Restart WebStorm

Right-click any `.html` file → **Share on HTML Drop**.

---

## Finder Quick Action (macOS)

1. Double-click `HTMLDrop.workflow`
2. Automator asks to install it — click **Install**

Right-click any `.html` file in Finder → **Services → Share on HTML Drop**.

> **Gatekeeper warning?** Right-click `HTMLDrop.workflow` → **Open**, then click **Open** again in the dialog.
